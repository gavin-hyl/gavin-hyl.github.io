Monte Carlo Tree Search with Thompson Sampling

Reinforcement Learning MCTS Bayesian Methods Python

I'm developing a novel variation of the Monte Carlo Tree Search (MCTS) algorithm that leverages Thompson sampling for more efficient and statistically motivated tree traversal and node expansion.

Traditional MCTS uses the UCB1 formula for the exploration-exploitation tradeoff, but my approach replaces this with Bayesian posterior sampling to more effectively handle uncertainty in the value estimates.

This project aims to improve the algorithm's performance in domains with high branching factors and noisy rewards, which are common challenges in robotics applications.

Customer Churn Prediction

Machine Learning Ensemble Methods Classification PyTorch

For this project, I developed a comprehensive ensemble model to predict whether customers would discontinue telephone service based on historical usage patterns and demographic data.

The ensemble combines multiple algorithms including:

  • Gradient Boosted Decision Trees (Adaboost)
  • Random Forests with optimized hyperparameters
  • Deep Neural Networks with custom architecture

This approach achieved 1st place in prediction accuracy in the course among 50+ competing models. The project demonstrates the power of combining different machine learning paradigms to improve predictive performance.

Shakespearean Poetry Generation

Natural Language Processing Hidden Markov Models Generative AI Python

I created and trained a hidden Markov model (HMM) that generates Shakespearean-style poetry by learning patterns from Shakespeare's sonnets.

The model works by:

  • Learning transition probabilities between hidden states
  • Modeling emission probabilities from states to words
  • Using the Baum-Welch algorithm for parameter estimation
  • Implementing a specialized sampling technique that maintains poetic meter

This project demonstrates how classical probabilistic models can still produce impressive results in generative text applications, even in the era of large language models.
